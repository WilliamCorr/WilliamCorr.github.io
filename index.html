My areas of interest

Software Design and Development

Throughout four years of university, I have made many small practical applications. Interested in all kinds of software development, having a background mostly in Java and Python. I have enjoyed working in teams employing design principles of the software development life cycle to create larger applications. The research component to software development is intriguing to me, trying to find newer and faster ways to implement solutions is always exciting.

Web Design and Development

Web design and programming for the web are a great challenge. I enjoy the constantly changing landscape of web design and development. My background in this area is in creating client/server applications in PHP to utilizing frameworks such as Symfony. The area of web programming that interests me the most is Python Flask when combined with cloud services such as Google App Engine and Amazon AWS. Currently building RESTful web services in Python.

Game and Simulation Development

Interactive multimedia and game design has always been a personal passion of mine. University work in this area was group work that followed producing a game design document and implementing that design in Unity. Having some experience with the Unreal engine, the Unity framework was chosen for familiarity in later projects and for its high interoperability with the windows operating system. Currently I am researching and designing games intended toward people with disabilities.

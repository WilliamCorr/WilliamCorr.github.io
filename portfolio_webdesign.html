Web Design & Development

Desktop/Mobile

Web design and development have always been the cornerstone of my education. Along with networking I have trained in HTML, CSS, and client-server programming in web development over the past four years. Beginning with the foundations of design in basic HTML and CSS, to more complex web environments with local databases. I have experience with Mobile first development with JQueryMobile using asynchronous callbacks and session memory control. On the Client-server programming side, I was trained in classful PHP with model view controller architecture and OOAD practices, moving later into frameworks such as Symfony implementing builder design patterns. During this period I completed a series of group web projects using an in-depth level of GitHub CLI, Linux CLI, and bash scripting.

Moving away from PHP into python and C++ for client-server programming. The experience I have in C++ is interacting with WiFi/Ethernet modules on Arduino hardware for IoT applications. A series of projects in python including writing IRC protocol using Sockets, Flask applications built on the Google App Engine, an Amazon EC2 instance running a custom-built WebService API using RabbitMQ passing commands between Python and PHP services.

Recently I have been experimenting with Web Assembly and the Rust programming language.
